📚 Documentation

Complete guide to using Funky Bot effectively

🚀 Getting Started

Step 1: Invite Funky Bot

Click the button below or visit our Invite Page to add Funky Bot to your Discord server.

📥 Invite Funky Bot

Step 2: Required Permissions

Ensure Funky Bot has these permissions for optimal performance:

  • ✅ View Channels
  • ✅ Send Messages
  • ✅ Embed Links
  • ✅ Connect to Voice Channels
  • ✅ Speak in Voice Channels
  • ✅ Use Slash Commands

Step 3: Basic Usage

🎵 Join a voice channel and use /play [song name] to start playing music!

🎛️ Command Categories

🎵 Music Playback Commands

  • /play [song] - Play a song or add it to queue
  • /pause - Pause the current track
  • /resume - Resume playback
  • /skip - Skip to the next song
  • /stop - Stop playback and clear queue
  • /nowplaying - Show currently playing song

📝 Queue Management

  • /queue - View the current queue
  • /clear - Clear the entire queue
  • /shuffle - Shuffle the queue
  • /remove [position] - Remove a song from queue
  • /move [from] [to] - Move a song in queue

🔊 Audio Controls

  • /volume [0-200] - Adjust playback volume
  • /bass [level] - Adjust bass boost
  • /nightcore - Apply nightcore effect
  • /vaporwave - Apply vaporwave effect
  • /8d - Enable 8D audio effect
  • /reset - Reset all audio filters

🎧 Playlist Commands

  • /playlist create [name] - Create a new playlist
  • /playlist add [name] [song] - Add song to playlist
  • /playlist load [name] - Load a playlist
  • /playlist delete [name] - Delete a playlist
  • /playlist list - View all your playlists

⚙️ Settings & Configuration

  • /settings - View current server settings
  • /prefix [new prefix] - Change bot prefix
  • /247 - Toggle 24/7 mode
  • /djrole [role] - Set DJ role for restricted commands
  • /language [lang] - Change bot language

✨ Features Guide

🎵 Multi-Platform Support

Funky Bot supports streaming from:

  • ▶️ YouTube - Videos and playlists
  • 🎧 Spotify - Tracks, albums, and playlists
  • ☁️ SoundCloud - Tracks and sets
  • 🍎 Apple Music - Links
  • 📻 Direct URLs - MP3, M4A streams

Usage: Simply use /play with a song name or URL!

⏰ 24/7 Mode

Keep Funky Bot connected to your voice channel all the time!

  • ✅ Bot stays connected even when alone
  • ✅ Automatically reconnects after disconnection
  • ✅ Perfect for music radio channels

Enable: Use /247 command (requires DJ role or admin)

🎛️ Advanced Audio Filters

Customize your listening experience with 15+ audio filters:

  • 🎸 Bass Boost - Enhance low frequencies
  • 🎹 Nightcore - Speed up and pitch up
  • 🌊 Vaporwave - Slow down and reverb
  • 🎧 8D Audio - Surround sound effect
  • 🎼 Karaoke - Remove vocals
  • ⚡ And many more!

🔄 Auto-Play & Smart Queue

  • 🎵 Auto-play suggests similar songs when queue ends
  • 🔀 Smart shuffle preserves queue diversity
  • 📊 Queue history tracking
  • 💾 Resume from where you left off

🔧 Troubleshooting

❓ Common Issues

🔇 Bot is not playing audio

  • ✓ Ensure bot has permissions to connect and speak
  • ✓ Check if you're in the same voice channel
  • ✓ Try using /stop and /play again
  • ✓ Verify server region compatibility

⚠️ Bot disconnects frequently

  • ✓ Enable 24/7 mode with /247
  • ✓ Check your server's voice region
  • ✓ Report persistent issues in support server

🐌 Slow response or lag

  • ✓ This may be due to high bot usage
  • ✓ Check Discord's server status
  • ✓ Try changing server region

❌ Commands not working

  • ✓ Ensure you're using slash commands (/)
  • ✓ Check bot permissions
  • ✓ Verify the command syntax

💡 Best Practices

🎯 Optimal Setup

  • 🎭 Set up a dedicated DJ role for music control
  • 🔒 Restrict certain commands to specific roles
  • 📢 Create a dedicated music channel
  • ⏰ Use 24/7 mode for radio-style servers
  • 📊 Regularly check bot status and updates

⚡ Performance Tips

  • 🎵 Use direct song URLs for faster playback
  • 📝 Create playlists for frequently played songs
  • 🔊 Keep volume between 50-100 for best quality
  • 🎛️ Avoid using too many filters simultaneously
